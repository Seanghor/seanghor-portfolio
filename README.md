# Seanghor Portfolio

Vue Js

## Run Locally

Clone the project

```bash
  https://github.com/Seanghor/seanghor-portfolio
```

Go to the project directory

```bash
  seanghor-portfolio
```

Install dependencies

```bash
  pnpm install
```

Start the Web Application

```bash
  pnpm run dev
```

Build Project

```bash
  pnpm run build
```
